# Storefront Media Slots

Este folder contiene solo media optimizada para la web publica. No copiar aqui la biblioteca original ni archivos pesados desde `data/biblioteca`.

## Videos reales aprobados para cocina

- `hero-cocina.mp4`: presenter + wok en cocina real.
- `detalle-wok.mp4`: tapa, interior granito, base y marca MGC.
- `receta-wok.mp4`: salteado de vegetales en wok.
- `uso-diario-gas.mp4`: olla en cocina a gas.
- `set-mgc.mp4`: set MGC completo sobre mesa.

## Legado / pendiente de reemplazo

- `photo-prueba-huevo.jpg`: no usar como promesa de huevo/queso hasta tener prueba real.
- `photo-limpieza-rapida.jpg`: no usar como promesa de limpieza hasta tener video real.
- `poster-huevo.svg` y `poster-limpieza.svg`: placeholders genericos de respaldo, no slots activos.

The storefront renders local JPG posters when MP4 files are not present, so the
page does not request missing video files or show broken media.

Do not copy third-party brand media into this folder unless the business has
usage rights. These files must be owned or approved for Eter Niu.

See `docs/CONTENT_MATRIX.md` for the source, section, product, CTA and CRM event
mapped to each media slot.
