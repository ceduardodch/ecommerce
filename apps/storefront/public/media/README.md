# Storefront Media Slots

Drop approved local media here for production:

- `hero-cocina.mp4`
- `prueba-huevo.mp4`
- `limpieza-rapida.mp4`
- `receta-wok.mp4`
- `editorial-mesa.svg`
- `editorial-set.svg`
- `editorial-cuidado.svg`

The storefront renders local SVG posters when MP4 files are not present, so the
page does not request missing video files or show broken media.

Do not copy third-party brand media into this folder unless the business has
usage rights. Use the existing local SVGs as temporary placeholders until the
real Eter Niu videos and photos are uploaded.
